# mock_test_website-
Note: This project is currently in the active building stage. I am currently transitioning the web architecture into a cross-platform mobile application.
Project Overview
This project is a web-based testing environment designed to simulate the actual GATE/ESE exam experience. It features a timed quiz interface, specific marking schemes  and an automated result evaluation system to help students assess their readiness.
Key Features (Currently Functional)
Marking Logic: Implemented GATE-specific scoring logic.
Automated Evaluation: Instant result generation with a breakdown of performance.
Tech Stack
Web: Node.js, Express.js, HTML5
Planned for App:Flutter,java/kotlin
Learnings
Scalability: Designing the backend to handle a growing database of technical questions.

User Experience (UX): Refining the interface to minimize cognitive load during high-pressure timed exams.

Full-Stack Integration: Managing data flow between the server and client in real-time.
